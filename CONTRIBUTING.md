# Contributing to Tactic Remote

Thank you for your interest in Tactic Remote!

## How to Contribute

Since this is a proprietary project, we do not accept code contributions at this time.

However, you can still help by:

### Reporting Bugs

1. Check if the issue already exists in [Issues](https://github.com/TacticSpaceTech/TacticRemote/issues)
2. Create a new issue with:
   - Clear description of the bug
   - Steps to reproduce
   - Expected vs actual behavior
   - Your environment (macOS version, iOS version, etc.)

### Suggesting Features

1. Open a [Discussion](https://github.com/TacticSpaceTech/TacticRemote/discussions)
2. Describe the feature and its use case
3. Explain why it would be valuable

### Feedback

We appreciate all feedback! Feel free to:
- Star the repository if you find it useful
- Share your experience in Discussions
- Spread the word about Tactic Remote

## Code of Conduct

Please read our [Code of Conduct](CODE_OF_CONDUCT.md) before participating.
